# HANDMATIC: Wireless Robotic Arm Control Through Gesture Recognition

## Overview
HANDMATIC is an embedded systems project that enables wireless control of a robotic arm using hand gestures. The system captures wrist and finger movements through a gyroscope and flex sensors, processes them using Arduino, and transmits control signals via RF communication. A NodeMCU-based receiver replicates the hand movements by controlling servo motors. Fingerprint authentication and keypad verification ensure secure access for authorized users.

## Features
- Gesture-based robotic arm control
- Wireless RF communication
- Flex sensor and gyroscope-based motion detection
- Fingerprint and keypad authentication
- Real-time LCD status display
- Servo motor-based robotic arm movement
- Battery-powered portable system
